# 🚨 Complaint Management System

A modern and responsive Complaint Management System built using **React.js** that allows users to register complaints, track complaint status, and manage issue resolution efficiently.

---

# 📌 Project Overview

The Complaint Management System is designed to simplify the process of submitting, tracking, and resolving complaints in organizations, colleges, offices, and institutions.

This project provides a user-friendly interface for users to raise complaints and an admin panel to manage and resolve them effectively.

---

# ✨ Features

## 👤 User Features
- User Registration & Login
- Submit Complaints
- View Complaint Status
- Track Complaint Progress
- Responsive User Dashboard
- Easy Navigation Interface

## 🛠️ Admin Features
- Admin Login
- View All Complaints
- Update Complaint Status
- Manage Complaint Resolution
- Organized Complaint Management

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/seshupavani/Complaint_Managment_System.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd Complaint_Managment_System
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run the Project

```bash
npm start
```
